# README

Este repositorio es de un TP de la materia Organización del Computador II.

El enunciado del TP original está en `enunciado/`.

El código se encuentra en `src/`.

Hay un informe del proyecto de migración a RISCV en `informe/`, renderizado en `informe.pdf`.

En `riscv/` está la adaptación del mismo proyecto a la arquitectura RISCV.

Hay resources de RISCV en `riscv-isa-manual/` y `learn/`.
